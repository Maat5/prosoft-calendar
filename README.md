### Prosoft calendar
___


### Installation

first clone the repository
```sh
$ git clone https://github.com/Maat5/prosoft-calendar.git
```

enter to cloned folder
``` sh
 cd prosoft-calendar
```

Next Install npm modules

```sh
$ npm install
```

Then install bower components

```sh
$ bower install
```

---

## Run
```sh
$  npm start
```
